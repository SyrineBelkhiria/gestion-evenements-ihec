# Gestion des événements – IHEC

Projet académique réalisé à l’IHEC en Java avec NetBeans.

## Description
Application permettant de gérer différents types d’événements (conférences, activités, etc.).

## Fonctionnalités
- Ajout d’un événement
- Modification d’un événement
- Suppression d’un événement

## Technologies
- Java
- NetBeans

## Objectif
Organiser et gérer efficacement les événements au sein de l’IHEC.
