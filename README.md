# Gestion des événements – IHEC

Projet académique réalisé à l’IHEC en Java avec NetBeans.

## Description
Ce projet porte sur la gestion des événements et a été réalisé dans le cadre du cursus à l’IHEC Carthage (2ème année Business Intelligence). Il permet de gérer des événements : création, modification, consultation et organisation (dates, lieux, participants, etc.). L’application vise à automatiser et structurer la planification d’événements dans un contexte académique ou professionnel. Le projet mobilise des compétences en conception de systèmes d’information, en modélisation des données et en développement d’applications.
## Fonctionnalités
- Ajout d’un événement
- Modification d’un événement
- Suppression d’un événement (etc)

## Technologies
- Java
- NetBeans

## Objectif
Organiser et gérer efficacement les événements au sein de l’IHEC.
